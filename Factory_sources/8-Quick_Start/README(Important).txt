[CN]
本产品不带板载USB转串口电路，Type-C接口只支持USB模拟串口，win10及以上系统可以直接使用，其他版本系统不能直接使用。

[EN]
This product does not come with an onboard USB to serial port circuit, and the Type-C interface only supports USB analog serial ports. 
It can be used directly on Windows 10 and above systems, but not on other versions of systems.
